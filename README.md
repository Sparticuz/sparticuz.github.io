# sparticuz.github.io

My personal site

### Dev Environment
 - https://help.github.com/articles/setting-up-your-github-pages-site-locally-with-jekyll/
 - To Install:
   - `https://jekyllrb.com/docs/installation/ubuntu/`
 - To Update (Ruby Dependencies):
   - `bundle update`
 - To Update (NPM Dependencies):
   - `npm update`
 - To Run:
   - `npm run build`
